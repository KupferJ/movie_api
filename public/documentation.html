myFlix Movie-App

This is the server-side component of a movie application, which will provide users with a variety of information (such as title, actors, genre, direction etc.) for a list of movies. The user will be able to sign up, edit personal information, and create and edit a list of their favourite movies.

Add a movie to user's favorite movies array /users/:Username/movies/:MovieID POST none A message indicating whether the movie was successfully added
